# SVG
- 확장 가능한 벡터 그래픽으로 XML 기반의 2차원 그래픽
- HTML 태그의 집합으로 CSS와 JS 로 컨트롤이 가능하다.
## SVG의 장점
- 벡터 이미지이므로 확대를 하더라도 이미지가 꺠지지 않는다.
- 이미지의 크기를 키워도 용량이 늘어나지 않는다.
## SVG의 단점
- 코드로 이루어진 이미지이기 때문에 복잡한 이미지일수록 파일 사이즈가 커진다.
- 단순한 모양일수록 효율이 좋다.
## svg 파일 사용
- 내부 조작이 불가능
    - img 태그 사용
    - css background 사용
- 내부 조작이 가능
    - inline으로 구현하기
    - Object 태그로 구현하기
